Apache Shiro 身份验证绕过漏洞 (CVE-2020-1957)

  • Apache Shiro 是一个功能强大且易于使用的 Java 安全框架，它执行身份验证、授权、加密和会话管理。

影响范围

  • Apache Shiro < 1.5.3

漏洞利用

  • 在具有 Spring 动态控制器的 1.5.2 之前的 Apache Shiro 版本中，攻击者可以使用构造恶意构造的请求 ..;绕过目录身份验证。
